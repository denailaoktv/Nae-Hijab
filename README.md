# Nae Hijab 🧕

**Anggun bukan soal mengikuti tren. Tapi menemukan versimu.**

Nae Hijab adalah website toko online (e-commerce) yang menghadirkan koleksi hijab voal, pashmina, instan, dan segiempat berkualitas premium dengan harga bersahabat. Dibangun dengan HTML, CSS, dan JavaScript murni (tanpa framework/backend), lengkap dengan simulasi alur belanja dari katalog hingga checkout.

## ✨ Fitur

- **Navbar responsif** dengan pencarian produk & menu hamburger untuk mobile
- **Hero section** dengan statistik toko
- **Katalog produk** dengan:
  - Filter kategori (Voal, Pashmina, Instan, Segiempat, Aksesoris)
  - Sortir produk (rekomendasi, harga termurah/termahal, nama A–Z)
  - Pencarian produk secara real-time
  - Modal detail produk
- **Keranjang belanja** (cart drawer) dengan subtotal, ongkos kirim, dan total
- **Checkout simulasi** lengkap dengan:
  - Form data pengiriman (nama, email, no. HP, alamat, kota, kode pos) + validasi
  - Pilihan metode pembayaran (Virtual Account, E-Wallet/QRIS, Kartu Kredit/Debit) — simulasi Midtrans Sandbox, belum ada transaksi nyata
  - Ringkasan pesanan & halaman sukses dengan kode pesanan
- Halaman **Tentang** (kurasi bahan, harga transparan, pengiriman cepat)
- **Footer** dengan info kontak & tautan bantuan

## 🛠️ Teknologi yang Digunakan

- **HTML5** — struktur halaman (`index.html`)
- **CSS3** — styling (`style.css`), font Google (Fraunces & Manrope)
- **JavaScript** — logika katalog, keranjang, dan checkout (`script.js`)
- **Google Analytics (gtag.js)** — tracking pengunjung
